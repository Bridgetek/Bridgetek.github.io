Create IDM2040-21R Projects
===========================

Using the internal ESD version (IDM2040-21R), you can create an ESD project that generates C code for the IDM2040-21R. After building the application, deploy it to the IDM2040-21R target platform.
To create an ESD IDM2040-21R project:

1. Select **File > New Project > Select option ID2040-21R  > Next**

.. image:: images/create_project.png

2. In configure project dialog, press **Configure**

.. image:: images/config_project.png
